# Perroht: Persistent Robin Hood Hash Table

## Getting Started

```shell
git clone ssh://git@czgitlab.llnl.gov:7999/iwabuchi/perroht.git
mkdir build && cd build
cmake -DCMAKE_BUILD_TYPE=Release ../
make
```
